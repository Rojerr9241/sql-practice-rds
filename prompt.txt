Objetivo del proyecto:
1. Guiarme para levantar una instancia RDS PostgreSQL en free tier 
   (incluyendo configuración correcta de security groups para conectarme 
   desde mi máquina local de forma segura).
2. Ayudarme a inicializar el repo con esta estructura, configurar uv, 
   .gitignore correcto, y un README claro.
3. Diseñar y cargar un esquema de datos relacional realista, basado en el 
   dataset Chinook (tienda de música: artistas, álbumes, clientes, 
   facturas, empleados). Si tienes una versión mejor o más completa para 
   practicar, sugiéremela.
4. Ayudarme a escribir progresivamente queries que cubran (en este orden): 
   JOINs (INNER, LEFT, y casos donde la diferencia importa), agregaciones 
   con GROUP BY/HAVING, subqueries y CTEs, y window functions (ROW_NUMBER, 
   RANK, LAG/LEAD).
5. Para cada bloque de queries, dame primero 2-3 preguntas de negocio 
   realistas (ej. "¿cuáles son los 5 clientes con mayor gasto total?") 
   ANTES de mostrarme la solución, para que yo intente resolverlas primero 
   y luego revisemos juntos mi versión vs. una versión optimizada.
